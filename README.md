## About

Social media platform for news and conversations.

## Features

- Timeline
- Tweets
- Retweets
- Likes
- Direct messages

## Installation

### From App Store
Open the **App Store** in zOS and search for "X".

### Via npm
```bash
npm install @zos-apps/twitter
```

## Usage

- Scroll timeline
- Click to view tweet
- Compose with + button



## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
